<h1 align="center"> Contact Management</h1>

<h2 align="left"> Descrição do Projeto </h2>

Este projeto foi desenvolvido com o modelo MVC usando o modelo de projeto Aplicativo Web do ASP.NET Core (Model-View-Controller).
O objetivo da aplicação é fazer um CRUD de gerenciamento de contatos, impedindo duplicidade.
O projeto foi desenvolvido usando Razor Pages, MariaDB (10.6), ASP.NET core, .Net 7.

<h2 align="left"> Pacotes Utilizados </h2>

- Microsoft.AspNetCore.Identity.EntityFrameworkCore
- Microsoft.EntityFrameworkCore.Tools
- Microsoft.EntityFrameworkCore.Design
- Pomelo.EntityFrameworkCore.MySql
